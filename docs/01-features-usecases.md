## ユースケース詳細

### UC-01: コンビニ惣菜のOCR登録

**概要:** コンビニで購入したサラダ等の栄養表を撮影し、カロリー・PFCバランスを登録する。

**フロー:**

1. 食事記録画面で「写真から登録」ボタンをタップ
2. カメラを起動し栄養表を撮影
3. OCRでテキストを抽出し、カロリー・タンパク質・脂質・炭水化物・商品名を自動解析
4. 抽出結果をフォームにプリセットする
5. ユーザーが内容を確認・修正し、価格を入力
6. 「登録」ボタンで食事記録に追加

**代替フロー:** OCR精度が低い場合、全項目を手動入力に切り替え可能。

---

### UC-02: 1日の食事記録と集計

**概要:** 朝食・昼食・夕食の食事を記録し、1日の合計カロリー・PFCバランス・食費を確認する。

**フロー:**

1. ホーム画面で対象の食事枠の「登録する」ボタンをタップ
2. メニューを追加（手動入力 / 食品マスタから選択 / レシピから選択 / セットメニューから選択 / OCR登録）
3. 各メニューにカロリー・PFC・食費を入力。レシピから追加する場合は1人分に自動算出され、何人分食べたか調整可能
4. 登録後、デイリーサマリーに合計値が自動反映
5. 登録済みの食事枠には「+ 追加する」ボタンが表示され、メニューの追加が可能

---

### UC-03: レシピ登録

**概要:** よく作る料理のカロリー・PFC・食費を計算してレシピとして保存しておく。

**フロー:**

1. レシピ一覧画面で「新規レシピ」をタップ
2. レシピ名・何人分かを入力
3. 材料リストを追加（材料名・分量・単価）
4. 完成品のカロリー・PFCを手動入力
5. 食費は材料の単価合計から自動計算
6. 保存後、食事記録時に「レシピから選択」で呼び出し可能

---

### UC-04: セットメニューの登録と利用

**概要:** 「カレー＋ご飯＋サラダ」などよく食べる組み合わせをセットメニューとして保存し、食事登録時にワンタップでまとめて記録する。

**登録フロー:**

1. セットメニュー画面で「+ 新規」をタップ
2. セットメニュー名を入力
3. レシピ・食品マスタからアイテムを追加
4. 各アイテムの分量を調整
5. 合計カロリー・PFC・食費が自動計算されて表示
6. 保存

**利用フロー:**

1. 食事登録画面で「セットメニューから選択」をタップ
2. セットメニュー一覧から選択
3. セットに含まれる全アイテムが食事記録に一括追加される

**ユースケース例:**

- カレー（レシピ）＋ 白ごはん200g（食品マスタ）→ セット「カレーライス」
- 鶏胸肉（レシピ）＋ サラダ1皿（食品マスタ）→ セット「鶏セット（サラダ）」
- トースト ＋ 目玉焼き ＋ サラダ → セット「朝食セット」

---

### UC-05: 1週間の献立計画

**概要:** 1週間分の献立を事前に決めておき、カロリーや食費の見通しを確認する。

**フロー:**

1. 献立画面で対象週を選択
2. 各曜日×食事タイミングのマスにメニューを割り当て
3. 週間合計カロリー・PFC・食費がリアルタイム表示
4. 献立確定後、当日の食事記録に献立メニューをワンタップで転記可能

---

### UC-06: 食事記録の編集と削除

**概要:** 登録済みの食事アイテムのカロリー・PFC・食費を修正したり、食べなかったアイテムを削除する。

**編集フロー:**

1. ホーム画面で修正したい食事アイテムをタップ
2. アイテム編集モーダルに、表示され全フィールドが編集可能
3. 内容を修正し「変更を保存」をタップ
4. デイリーサマリーがリアルタイムで再計算される

**削除フロー:**

1. アイテム編集モーダル下部の「このアイテムを削除する」をタップ
2. 削除確認ダイアログが表示される
3. 「削除」をタップするとアイテムが削除され、トーストで「削除しました（元に戻す）」が表示

---

### UC-07: データエクスポート

**概要:** 食事記録・レシピ・食品マスタのデータをCSVファイルとしてエクスポートする。

**フロー:**

1. 設定画面でエクスポートしたいデータ種別を選択（食事記録 / レシピ / 食品マスタ / セットメニュー）
2. 期間指定（食事記録の場合）またはすべてのデータを選択
3. CSVファイルが生成され、ブラウザのダウンロードが開始

**備考:** 優先度は低。Phase 5（改善・拡張）で実装予定。
